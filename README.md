# Paperless Pulse
Clone the repo: 
```
git clone https://github.com/yourusername/PaperlessPulse.git
```

Install dependencies:
```
npm install
```

Create local .env: They must manually create a .env in the server folder with their own PostgreSQL credentials.
```
DB_USER="Your_Username"
DB_PASSWORD="Your_Password"
DB_HOST="localhost"
DB_PORT="5432"
DB_NAME="PaperlessPulse"
JWT_SECRET="Your_secret_key"
PORT="5000"
```
Run the App:
```
npm run dev 
```
(in client)
```
node server.js 
```
(in server)

To run the client & server concurrently, type below command in root directory 
```
npm start
```
